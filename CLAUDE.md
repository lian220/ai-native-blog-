# Claude Code 프로젝트 규칙

## 언어 규칙

- PR(Pull Request) 제목과 본문은 **한글**로 작성합니다.
- 커밋 메시지는 영어로 작성하되, 필요시 한글 설명을 추가할 수 있습니다.
- 코드 주석은 한글 또는 영어로 작성할 수 있습니다.

## PR 생성 규칙

PR 생성 시 다음 형식을 따릅니다:

### 제목
- `feat: 기능 설명` - 새로운 기능
- `fix: 버그 설명` - 버그 수정
- `refactor: 리팩토링 설명` - 코드 개선
- `docs: 문서 설명` - 문서 수정

### 본문
```markdown
## 요약
- 변경사항 1
- 변경사항 2

## 테스트 계획
- [ ] 테스트 항목 1
- [ ] 테스트 항목 2
```

## 기술 스택

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- pnpm
